Why Choose Polystate?

📦

Code Generation

Define your store once. Automatically generate Redux or NgRx code with zero manual edits.

🔄

Framework Agnostic

Write once, use with React 18+, Angular 17+, or vanilla JavaScript. Your choice.

⚡

Zero Dependencies

@polystate/core has ZERO runtime dependencies. Lightweight and blazing fast.

🔒

Type Safe

Full TypeScript support in strict mode. Compile-time action validation. No runtime surprises.

🛠️

DevTools Ready

Redux DevTools integration with time-travel debugging. Perfect for enterprise development.

📊

Middleware System

Built-in middleware for logging, persistence, custom side effects. Extensible architecture.

Two Ways to Use Polystate

📝 Code Generation

Perfect for production apps

  • Define store.definition.ts
  • Run polystate generate
  • Get Redux/NgRx code
  • Type-safe dispatch
  • Ready to ship

⚙️ Runtime Adapters

Perfect for prototyping

  • createStore() with actions
  • Use React hooks or Angular services
  • No build step needed
  • Fast iteration
  • Seamless debugging
