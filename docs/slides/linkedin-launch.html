The Solution

Define once.
Use anywhere.

⚡

Runtime Mode

Drop @polystate/core directly into React or Angular. No store setup, no providers — just hooks and services on top of a zero-dependency reactive core.

🛠

Generate Mode

Run the CLI and get fully typed, idiomatic Redux + RTK (React) or NgRx (Angular) code. Your team keeps their workflow — you write the logic once.

See it in action

Define once, generate for both.

// store.definition.ts
export const todoDefinition = {
  name: 'todo',
  initialState: { todos: [] },
  actions: {
    addTodo: (state, title) => ({
      ...state,
      todos: [
        ...state.todos,
        { title, done: false }
      ]
    })
  }
}
# Generate native Redux
polystate generate \
  store.definition.ts --react

# Generate native NgRx
polystate generate \
  store.definition.ts --angular
